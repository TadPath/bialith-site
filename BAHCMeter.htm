Histocytometer

Histocytometer is an MS Windows GUI program designed to allow the user to quantify cells in digital images by interactive counting and basic stereology.

The program brings together some commonly used image processes such as background shading correction, thresholding and interactive differential cell counting to allow for numbers and proportions of different cell types (other other objects if using non-bioloical images) to be made.

I originally wrote the software to do percentage counts of plasma cells or blasts in bone marrow trephine specimens and Ki-67 labelling index measurements in tumours, etc.

See the Downloads page to get a copy of the program as well as the detailed user manual.
